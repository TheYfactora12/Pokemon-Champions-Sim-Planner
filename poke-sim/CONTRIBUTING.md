# Contributing to Pokemon Champions 2026 Sim

This guide is for people contributing directly to the `poke-sim/` app.

## Setup

1. Install Git: https://git-scm.com/downloads
2. Install VS Code: https://code.visualstudio.com
3. Clone the repo
4. Open the project in VS Code

## Daily workflow

1. Start from `main`
2. Create a branch for one task
3. Make the change in `poke-sim/`
4. Open `poke-sim/index.html` in a browser to check the UI
5. Use the browser console for runtime errors

## Before you push

1. Run the relevant tests
2. Rebuild `poke-sim/pokemon-champion-2026.html` after source edits
3. Confirm the bundle and source match

## Submit

1. Commit the change with a clear message
2. Push the branch
3. Open a pull request
4. Assign Alfredo as reviewer

## Rules

- Keep each branch small
- Do not push directly to `main`
- If something fails, capture the error and fix it before merging
